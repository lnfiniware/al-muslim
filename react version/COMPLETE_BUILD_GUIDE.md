## 📱 INSTALL ON ANDROID PHONE

### Before Installing

On your Android phone:

1. Go to **Settings**
2. Go to **Apps**
3. Go to **Special App Access** (or App Permissions)
4. Go to **Install Unknown Apps** (or Similar)
5. Select your **File Manager**
6. Enable it (turn ON) ✅

### Install Steps

1. **Transfer APK to phone:**
   - USB cable (connect to phone)
   - Or email to yourself
   - Or upload to Google Drive/OneDrive
   - Or any other method

2. **Open file manager** on phone

3. **Find the APK file** (almuslim.apk)

4. **Tap the file** to start installation

5. **You'll see:**
   ```
   Install "Almuslim"?
   - This app is not verified
   ```
   
   **Tap "Install"** (it's safe!) ✅

6. **Grant permissions when asked:**
   - Location (for prayer times)
   - Notifications (for reminders)
   - Camera (if needed)

7. **After installation:**
   ```
   Installation complete
   □ Open
   ```
   
   **Tap "Open"** to start app! 🎉

## 🐛 TROUBLESHOOTING

### Build Won't Start

**Problem:** "Build failed" or "No credentials"

**Solution:**
```bash
eas credentials
eas build --platform android --profile production
```

### APK Won't Download

**Problem:** Download link not appearing

**Solution:**
```bash
eas build:list
```

This shows all your builds with download links.

### Installation Fails

**Problem:** "App not installed"

**Solution:**
- Uninstall any old version first
- Enable "Unknown sources" in phone settings
- Try transferring via USB instead
- Restart phone and try again

### App Crashes on Launch

**Problem:** App opens then closes

**Solution:**
1. Grant all permissions
2. Check phone has internet
3. Clear app data and reinstall
4. Check Android version is 8.0+
