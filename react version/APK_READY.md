## 🚀 BUILD APK IN 3 STEPS

### Step 1: Install EAS CLI (one time only)
```bash
npm install -g eas-cli
```

### Step 2: Login to Expo
```bash
eas login
```
(Create free account at https://expo.dev if needed)

### Step 3: Build the APK
```bash
cd "e:\Zyad\Scripts\Ma shitty projects\almuslimphone"
eas build --platform android --profile production
```

**That's it!** ⏱️ The build takes 10-20 minutes. You'll get a download link when done.

## 📱 Install on Phone

1. **Download** the APK from the link shown
2. **Transfer** to your Android phone (email, USB, cloud storage)
3. On phone: **Settings** → **Apps** → **Special App Access** → **Install Unknown Apps** → Enable for your file manager
4. **Open** file manager and tap the APK file
5. **Install** → **Grant permissions** (Location & Notifications)
6. **Launch** the app! 🎉
