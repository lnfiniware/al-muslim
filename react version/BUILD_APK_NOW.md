## Step 4️⃣: Build the APK (15 minutes)

```bash
cd "e:\Zyad\Scripts\Ma shitty projects\almuslimphone"
eas build --platform android --profile production
```

**What happens:**
- Builds in the cloud ☁️
- Shows progress: 0% → 100%
- Takes 10-20 minutes
- Shows download link when done ✅

**See download link?** ✅ Move to step 5

---

## Step 5️⃣: Install on Phone (3 minutes)

**On your Android phone:**

1. **Enable Unknown Sources:**
   - Settings → Apps → Special App Access → Install Unknown Apps
   - Enable for your file manager

2. **Download APK:**
   - Click the download link from terminal
   - Or transfer the APK file to phone

3. **Install:**
   - Open file manager
   - Find the APK file
   - Tap it to install

4. **Grant Permissions:**
   - Location (for prayer times)
   - Notifications (for reminders)
   - Click "Install" when prompted

5. **Launch:**
   - Tap "Open" or find "Almuslim" in apps
   - Select language (English or العربية)
   - Enjoy! 🎉

## 📞 Need Help?

**Build won't start?**
- Ensure you're logged in: `eas whoami`
- Check internet connection
- Try: `eas build --platform android --profile production --wait`

**APK won't install?**
- Enable "Unknown sources" in phone settings
- Uninstall old version first
- Restart phone and try again

**App crashes?**
- Check all permissions granted
- Clear app cache
- Reinstall
